## Documentatie proiect T1

## Rolul directoarelor din proiect: 

### `bin/` - aici am programele executabile

### `src/` - codul sursa principal

### `include/` - aici tin fisierele header din c++ pentru folderul src/

### `data/` - aici fisierele statice (.in de ex) esentiale pt program

### `logs/` - salvez logs urile (cand a fost folosit programul)

### `reports/` - aici salvez rezultatele pe care le obtin dupa ce rulez programul

### `tmp/` - aici programul isi va crea singur fisiere folosite temporar pentru asi incheia sarcina/calculele stergandule la final

### `tests/` - scriptu care imi testeaza codul din src/ cand fac schimari

### `doc/` - aici salvez instructiunile si spun ce fac (documentatia)

### `tools/` - salvez scripturi mici care ajuta la gestionare

## Cum se ruleaza:

### Din proiect bash tools/t1_audit.sh

## Unde sunt salvate rapoartele:
### Rezultatele si rapoartele vor fi salvate in directorul "reports" si in subdirectoarele acestuia ("reports/fs", "reports/process", "reports/proc", "reports/pipeline")
